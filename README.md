🐉 Dragon Cursor Project

A simple and interactive cursor animation project built using HTML and JavaScript.

---

🚀 Live Demo

👉 https://MR-xPYTHONx.github.io/dragon-cursor/

---

✨ Features

- 🐉 Custom animated cursor
- 🎯 Smooth movement and effects
- ⚡ Lightweight and fast

---

📁 Project Structure

- "index.html"→ Main HTML file

- "main.js" → JavaScript logic

- "package.json" → Project configuration

---

▶️ How to Run

1. Download or clone this repository
2. Open "index.html" in your browser

---

📦 Install Dependencies (if needed)

npm install

---

🛠️ Technologies Used

- HTML
- JavaScript

---

📌 Notes

- "node_modules" is not included (to keep repo lightweight)
- Install dependencies using "npm install"
